## Come Deployare

### Opzione 1: GitHub Pages (Attuale)
1. Vai alla repository GitHub del sito
2. Sostituisci tutti i file nella cartella `dist/` con quelli ottimizzati
3. Fai commit e push
4. Il sito si aggiornerà automaticamente

### Opzione 2: Dominio Personalizzato (Raccomandato)
1. Acquista il dominio `nevenoir.com`
2. Configura DNS per puntare a GitHub Pages
3. Aggiorna i file con il nuovo dominio:
   - `index.html`: Aggiorna canonical URL
   - `sitemap.xml`: Aggiorna tutte le URL
   - `robots.txt`: Aggiorna Sitemap URL
4. Deploya i file aggiornati
